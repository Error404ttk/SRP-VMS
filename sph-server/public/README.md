# SPH Vehicle Log - Static Assets Directory

Place your static files (such as images, custom logos, or favicons) in this directory.

* **Favicon:** Rename your favicon file to `favicon.ico` and place it here to resolve the 404 favicon error.
* **Logos:** Place your system logo (e.g., `JPEG.JPG` or `logo.png`) in this folder. You can reference them inside the Google Sheet's `settings` under the `logo_url` column as `/JPEG.JPG` or `/logo.png`.
